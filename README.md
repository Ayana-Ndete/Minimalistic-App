# Minimalistic App

**ScrollStop** is a minimalist Android application designed to help users reduce doomscrolling by limiting the amount of time spent on distracting applications such as Instagram, TikTok, YouTube, Facebook, and others.

The app encourages healthier digital habits by allowing users to select apps, set daily usage limits, and receive reminders when their allotted time has been reached.

---

## ✨ Features

- Select apps to monitor
- Set daily usage limits
- View selected app and usage information
- Simple and clean minimalist interface
- Designed using Jetpack Compose
- Built entirely in Kotlin

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest version)
- Android SDK
- Android device or emulator
- Kotlin

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/ScrollStop.git
```

2. Open the project in Android Studio.

3. Allow Gradle to sync.

4. Run the application on an emulator or Android device.

---

## 📖 How It Works

1. Open the application.
2. Select the app you want to limit.
3. Set a daily time limit.
4. Start monitoring.
5. ScrollStop tracks usage.
6. When the limit is reached, the app will notify the user and encourage them to take a break.

---

## Project Goal

The objective of ScrollStop is to promote mindful smartphone usage by helping users become more aware of how much time they spend on distracting applications.

Unlike traditional app blockers, ScrollStop aims to encourage intentional usage through a simple, distraction-free user experience.
